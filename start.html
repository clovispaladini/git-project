O software:

Para adquirir o software, com versões para Mac OS, Windows e Linux/Unix, basta baixar o Git gratuitamente no site ofifial - https://git-scm.com/

Após o programa ser instalado, executar o terminal Git Bash.

Configuração inicial

Vamos começar a configurar o Git. Com o terminal do Git Bash aberto, siga as instruções abaixo:

Verificar a instalação do Git

Agora vamos iniciar a configurar o Git. O primeiro comando a ser executado é o git, com ele veremos se a instalação ocorreu normalmente e se o Git esta pronto para ser inicializado. Se tudo ocorreu bem, ira aparecer uma mensagem parecida com o exemplo abaixo:

Com o programa instalado corretamente, seguiremos para o próximo passo.

Configurar o seu nome no Git

Com o comando git config --global user.name "Seu Nome" cadastramos o nome do desenvolvedor no Git.

    
    usu@HOST MINGW64 ~
    $ git config --global user.name "Fernando Machado"   

    usu@HOST MINGW64 ~
    $

Configurar o seu email no Git

Com o comando git config --global user.email "seu.email@gmail.com" cadastramos o email do desenvolvedor.

    
    usu@HOST MINGW64 ~
    $ git config --global user.email "seu.email@gmail.com"   

    usu@HOST MINGW64 ~
    $

Habilitar as cores no terminal do Git Bash

Com o comando git config --global color.ui true habilitamos as cores no terminal do Git.

    
    usu@HOST MINGW64 ~
    $ git config --global color.ui true   

    usu@HOST MINGW64 ~
    $

Iniciar o Git no repositório do projeto

Após ser efetuada as configurações iniciais, agora vamos efetivamente instalar o git no repositório do projeto.

Dicas iniciais:

  • Comando ls lista as pastas do repositório.
  • Comando cd pasta entra no diretório.
  • Comando cd volta para o diretório raiz.
  • Comando rm -rf diretorio/ apaga todo diretório.
  • Comando clear limpa a tela do terminal.
  • Comando exit fecha o terminal do Git Bash.
  • Digitando as primeiras letras e clicando a tecla tab irá retornar o diretório com nome inicial dessas letras.
Acessando a pasta raiz do projeto

Iremos iniciar o Git. Nos exemplos que serão mostrados a partir de agora, a pasta raiz do projeto que será controlado está em uma pasta chamada 'git-project' na área de trabalho do meu computador. Vamos acessar este diretório conforme o exemplo abaixo:

    
    usu@HOST MINGW64 ~
    $ cd Desktop
    
    usu@HOST MINGW64 ~/Desktop  
    $ cd git-project

    usu@HOST MINGW64 ~/Desktop/git-project   
    $
Iniciando o Git

Dentro do diretório do projeto, executar o comando git init.

    
    usu@HOST MINGW64 ~/Desktop/git-project
    $ git init
    Initialized empty Git repository in C:/Users/Desktop/git-project/.git/   

    usu@HOST MINGW64 ~/Desktop/git-project (master)
    $

Pronto. O primeiro passo foi dado... a partir de agora, tudo que for modificado dentro do repositório 'git-project' será controlado pelo sistema Git.
